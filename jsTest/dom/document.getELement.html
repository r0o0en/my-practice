document.getElement

获取页面中的元素（dom对象）

常用发放

getElementById(id)
通过 id 获得相同 id属性的第一个元素。
getElementsByName(name)
通过元素的 name属性 来获取对应元素集合
getElementsByTagName(tagName)
通过tagName(标签名称)获取对应的元素集合。
getELementsByClassName(class)
通过class(类名)获取对应的元素集合
querySelector(cssSelector)
通过css选择器获取对应的元素中的第一个元素
querySelectorAll(cssSelector)
通过css选择器获取对应的元素集合
docuemnt. 参数 返回值 兼容性
getElmentById id属性 domObject
getElementsByName name属性 array 伪属性（IE8及以下）
getElementsByTagName 标签名称 array
getElementsByClassName 类名 array IE8（包括）↓
querySelector css选择器 domObject IE7（包括）↓
querySelectorAll css选择器 array IE7（包括）↓

伪属性：div、span在标准文档中没有规定必须有Name属性的元素，它们的name属性就是伪属性。

getElementById(id)

通过id属性来获取对应的元素，且：无论页面上有多少个相同ID的元素存在，都只会获取第一个满足条件的元素。

					
<ul>
	<li id="testa">1</li>
	<li id="testa">2</li>
	<li id="testa">3</li>
	<li id="testa">4</li>
</ul>
<script type="text/javascript">
	document.getElementById('testa').innerHTML; //echo 1
</script>

getElementsByName(name)

通过元素的 name属性 来获取对应元素

getElementsByName 在部分低版本浏览器中不支持伪属性

					
<ul>
	<li nama="testName">1</li>
	<li name="testName">2</li>
	<li name="testName">3</li>
	<li name="testName">4</li>
</ul>
<script type="text/javascript">
	document.getElementsByName('testName')[1].innerHTML;//echo 2 
</script>

getElementsByTagName(tagName)

通过元素的标签名称来获取对应的标签元素

					
<ul>
	<li >1</li>
	<li >2</li>
	<li >3</li>
</ul>
<script type="text/javascript">
	document.getElementsByTagName('li')[2].innerHTML; //echo 3
</script>

getELementsByClassName(class)

通过类名来获取对应的元素

					
<ul>
	<li class="test-class">1</li>
	<li class="test-class">2</li>
	<li class="test-class2">3</li>
	<li class="test-class">4</li>
</ul>
<script type="text/javascript">
	document.getElementsByClassName('test-class')[2].innerHTML; //echo 4	
</script>
