call() 和 apply()

在 Function.prototype中，有call()和apply()两个方法,它们可以：使一个函数使用一个指定的this进行运算。


//现有对象 a 和 b 
var a = {
	x:9,
	y:10
},
b = {
	x:1,
	y:2,
	add:function(){
		console.log( this.x + ' + ' + this.y +' = ', this.x + this.y );
	}
};
//其中对象B有方法 add
b.add(); // 输出 1 + 2 =  3；
//Function.prototype.call
b.add.call(a);  //输出9 + 10 =  19  这是同样运行了b.add方法，但是call(a)使b.fun执行时其作用域中的 this = a
//function.prototype.apply
b.add.apply(a); //输出 9 + 10 =  19

Function.prototype.call

call()方法调用一个函数 fun，使fun具有 一个指定的this值 和分别提供的参数（参数的列表）。

fun.call(thisArg,arg1,arg2,...)
